# Code of Conduct

This project is trying to become a community-viable ArduPilot configurator. Keep discussion technical, respectful, and focused on moving the product forward.

## Expected Behavior

- critique ideas, code, and product decisions directly without attacking people
- assume good faith while still holding a high engineering bar
- keep bug reports, reviews, and design feedback specific and actionable
- be careful with live-aircraft advice and hardware safety claims

## Unacceptable Behavior

- harassment, personal attacks, or discriminatory behavior
- aggressive gatekeeping or bad-faith nitpicking
- unsafe advice presented as verified fact
- sharing private or security-sensitive information without consent

## Scope

This applies to repository discussions, issues, pull requests, and any linked community spaces used for project coordination.

## Reporting

If someone violates this code of conduct, contact the maintainer privately. Security-sensitive or safety-critical concerns should follow [SECURITY.md](SECURITY.md).
